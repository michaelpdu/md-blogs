### L0/L1/L2范式
```
L0范数是指向量中非0的元素的个数。
L1范数是指向量中各个元素绝对值之和。
L2范数是指向量各元素的平方和然后求平方根

既然L0可以实现稀疏，为什么不用L0，而要用L1呢？
个人理解：一是因为L0范数很难优化求解（NP难问题），二是L1范数是L0范数的最优凸近似，而且它比L0范数要容易优化求解。
所以大家才把目光和万千宠爱转于L1范数。
```

### 正则化
```
L1 norm 是指对样本的每一个元素都除以该样本的L1范数. 
L2 norm 是指对样本的每一个元素都除以该样本的L2范数. 

sklearn中还有一种常用的normlize的方法z-score，也就是我们常用的 减去样本均值除以样本方差。
L1和L2范数是样本内的操作，而z-score是样本间的，将每个样本的第一个变量之间取均值，除以方差。
```
